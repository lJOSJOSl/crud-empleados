# 👷 CRUD de empleados

Aplicación desarrollada
en Python que crea una base
de datos, simulando la planilla
de trabajadores en una empresa.

Permite crear empleado, ver
planilla de empleados, actualizar
datos de los empleados y eliminar
empleados de la base de datos.

## 🔧 Funcionalidades:
- Crear empleado
- Ver empleados
- Actualizar datos
- Eliminar empleados

## 📚 Estructura del proyecto:

- base_de_empleados.py → Archivo principal del programa  
- README.md Acerca del proyecto

## 📡 Tecnologías utilizadas
- Python
